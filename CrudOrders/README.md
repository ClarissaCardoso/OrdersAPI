# Orders API

API REST para gerenciamento de pedidos.

## Tecnologias

- Node.js
- Express
- PostgreSQL

## Estrutura

Controller → Service → Repository → Database

## Instalação

1. Clonar repositório

git clone https://github.com/SEUUSUARIO/orders-api.git

2. Instalar dependências

npm install

3. Criar banco

CREATE DATABASE ordersdb;

4. Executar script

database/init.sql

5. Rodar aplicação

node server.js

Servidor rodará em:

http://localhost:3000
